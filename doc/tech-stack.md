# Technical Stack Decisions

This document records only explicitly confirmed technology choices for this project.

## Confirmed Stack

- Language: TypeScript
- UI library: React
- Application framework: Next.js
- Package manager: pnpm
- Deployment: Docker-based deployment
- Routing: Next.js App Router
- Source layout: `src` directory
- Styling: Tailwind CSS
- Component system: shadcn/ui
- Icon library: lucide-react
- UI state for v1: React local state
- Quality gates: ESLint, Prettier, Playwright
- Docker runtime: Next.js standalone output

## Decision Policy

Do not add unconfirmed technology choices to this document. Record new stack decisions only after they are explicitly confirmed.
